# Releases Data

Metadados consolidados de todas as releases do ecossistema The Bootstrap. Usado como fallback e referência centralizada.

## Publicar

1. Gere o `releases.tar.gz`
2. Crie release no GitHub com tag `releases-data-v0.0.X`, asset: `releases.tar.gz`
3. Atualize `resonance.json` no repositório `the-bootstrap`
